Internet

A computer network system which is connected throughout the world.

Web

An information system people who connected through the internet can exchange information.

How are the Internet and Web different?

Internet means computer network while the Web is one of Internet services.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange, and is character encoding. Unicode is an indusrty standard designed all characters in the world are represented on computers.

What is UTF-8

UTF-8 stands for UCS Transformation Format 8, and is the most common character encoding on the WWW.

URL

URL stands for Uniform Resource Locator, and is a web address that provides a specific location for a specific resource in the internet.

Domain Name

is a name which is easy to remember associated with a real IP address on the internet.

HTTP

HTTP stands for Hypertext Transfer Protocol, and is a set of communication rules for communicating between a client and a server.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure, and is more secure version of HTTP.

IP Address

A unique number used to distinguish each device connected to the internet.

DNS

DNS stands for Domain Name System, and it converts the domein to an IP that can be found on an IP network.

W3C Recommendations

A standard certified by the World Wide Web Consortium.

XML, XHTML, HTML, HTML5

XML stands for Extensible Markup Language, and is a markup language that provides rules to define data. XHTML stands for Extnsible Hyper Text Markup Language, and is a term used to describe HTML documents written to conform to XML syntax rules. HTML stands for Hyper Text Markup Language, and is the dominant markup language developed for displaying web pages. HTML5 is a web programming language that has evolved from displaying only text and hyperlinks to express and provide various applications including multimedia.
