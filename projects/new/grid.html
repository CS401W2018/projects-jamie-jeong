What should you focus on during class and after class?

It is important to pay attention during class because it helps us to follow the course well. After class, we can practice what we have learned.

What is the main difference between CSS Grid and CSS Flex?

CSS Grid is 2-dimensional, and can control both rows and columns. It is good for full page layouts. CSS Flex is 1-dimensional, and can control either rows or columns, not at the same time. It is good for aligning items in a single row or column, such as navigation bars.
