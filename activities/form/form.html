The form tag has an "action" attribute. What does it do?

The action attribute is a form tag specifies the URL where the form's data will be sent when the user submits it.

The form tag has a "method" attribute. What does it do?

The method attribute defines how the form data should be sent to the server. It can be either GET or POST.

What is the difference between a "POST" and a "GET"?

GET is sent in the URL, and the data is visible in the URL while POST is sent in the request body, and the data is hidden. Use GET for searching and POST for sensitive data like passwords.
