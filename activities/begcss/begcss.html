Engagement-Beginning CSS

Explain the difference between inline, embedded, and external CSS.

Inline CSS
The style attribute inside the tag is used to apply the style directly, affecting only that element.
Embedded CSS
Styles are applied to an entire page by writing them in the style tag inside the HTML head.
External CSS
You can write styles in separate .css files and link to them in your HTML to reuse them on multiple pages.

Explain why you should only use external CSS.

  1. Separation of concerns ensures that information is structured and presented correctly.
  2. Using a single stylesheet for multiple web pages helps maintain a cohesive look and consistency. For example, a university website can ensure visual uniformity across different department pages.

What is a selector used for?

Selectors are responsible for applying styles to specific HTML elements in CSS by specifying them.
