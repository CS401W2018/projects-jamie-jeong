Why is the viewport meta tag important?

The viewport meta tag makes web pages look good on different screen sizes. Without it, a site might appear too small or zoomed out on mobile devices. It helps create a responsive design.

How can you adjust the CSS for printing?

I can create CSS for print and use rule like media="print" to set styles for printing. For example, I can hide a navigation tab and change color or margin.

When would you want to adjust the CSS for printing?

I would adjust CSS for printing when I want a webpage to print so I can make the page clear and well formatted without extra clutter.
