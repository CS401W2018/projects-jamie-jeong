Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link is the full URL, starts with 'https://(domain)/(path)',and works anywhere. A relative link works within the same website, and is a shorter path. Use relative links because they are flexible and easier to update.

List the 3 rules with relative links.

  1. Same folder, use the file name.
  2. Subfolder, if the file is in a subfolder, include the folder name.
  3. To link to a file in the parent folder, use '../' to go up one level.
