When do you use a table?

When we organize the data in rows and columns, and in case of we need the structure that is convenience to compare.

When do you not use a table?

When we create a layout and need to just align texts.

When do you need to do extra work for table accessibility?

Extra work is needed when a table is complex, contains form inputs, is used for layout, needs screen reader support, or requires a description.

Why don't we use the border attribute?

Border attribute is outdated. Now we use CSS for better flexibility and design.

What is the easiest way to color every other row a different color?

The easiest way is to use 'nth-child(ever)' or 'nth-child(odd)'.
